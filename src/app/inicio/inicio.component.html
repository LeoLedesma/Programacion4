Clase 1

  • [(NgModel)]: Se llama doble bindeo o Two Way Binding, donde la comunicacion es bidireccional, cambiando el valor del atributo, tanto en la interfaz como en el modelo. Dividiendo lo que es cada elemento queda asi:
  • [Property Binding]: Ejemplo [value] = "nombre" -> Donde value es la propiedad de un elemento HTML. Modelo -> Vista
  • (Event Binding): Ejemplo (click)="sumar()" -> Donde al lanzarse el evento click, ejecuta el codigo javascript entre las comillas.
